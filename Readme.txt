# Front-end # 
- компиляцию выполняем инструментом gulp, запуская его с корневой директории проекта. 
Инструмент выполняет компиляцию стилей, сжатие в один файл, дописывани префиксов, соединение js скриптов в один файл и их минификацию.
- сжатие новых картинок выполняется командой(задачей) gulp images-min. 
- также есть возможность запустить gulp в режиме слежки за файлами - gulp dev. 
При каждом изменении файлов стиля или javascript файла будет автоматически перезагружена, также выполнена компиляция без сжатия. Реализуется слежка через [browser-sync](https://www.browsersync.io/). 
Возможно также тестировать верстку, проверять сценарии сразу в нескольких браузерах, для этого необходимо открыть сайт в любом другом браузере.
- документация по правильному написанию SASS и по сооблюдению стиля находится здесь -https://sass-guidelin.es/ru/

**После любых правок, обязательно пересобирать проект командой gulp.**

## Path ##

- файлы стилей SASS - assets/css/
- javascript файлы находятся в assets/js/
- в папке public/javascripts, public/stylesheets находятся уже скомпилированные файлы, изменять их нужно только посредством компиляции.

### Установка зависимостей ###

terminal:
> npm i


Режим слежки, разработки:
> gulp dev

Сборка проекта
> gulp
