The widget shows a popular entry in your blog. 

== Description ==

EN: The widget will show the most popular on your blog, basing on the number of comments to the recordings.
RU: Виджет покажет популярные записи вашего блога, основываясь на количестве комментариев к записям.

Features:

*   configurable number of displayed records;
*   conclusion quote, or just the title;
*   display in a list or any other form (the tags are set);
*   you can set the display of posts from only certain categories;
*   you can specify whether or not the number of comments.

== Installation ==

1. Unzip files.
2. Upload 'wp-popularposts' to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Activate the widget through the 'Widgets' menu in WordPress

== Frequently Asked Questions ==

= Can I customize the look of the widget? =

Yes, look, you can customize the output to record headers, or headers and citations. Also set up html tags to each entry.
